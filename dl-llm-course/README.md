# 深度学习与大模型安全 · 课程根目录

这是你的学习工作区。所有笔记、实验、进度都在这里，纳入 Git 管理。

## 怎么用
- 每节课一个文件夹，编号即课号（如 `01_foundations/02_pytorch_autograd/`）。
- 学习进度看 [PROGRESS.md](PROGRESS.md)——tutor 每次开课先读它。
- 每节课文件夹里有 README（目标/前置/检查点）、lab（实验）、quiz（掌握小测）。
