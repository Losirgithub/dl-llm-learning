> 每节课结束时由 tutor 生成，固化推导、公式、以及学员的误解与正确答案。

## 一、关键公式

**MLE 框架**
- 似然 L(θ) = Π P(xᵢ | θ)  （各点概率连乘）
- log 似然 log L(θ) = Σ log P(xᵢ | θ)  （取 log 变连加，好求导）
- 求导令 = 0，解出 θ

**正态分布（高斯）**
P(x) = (1/√(2πσ²)) · exp( -(x-μ)² / (2σ²) )
- μ：钟形中心位置；σ²：离散程度（σ 大=矮胖分散，σ 小=高瘦集中）

**伯努利分布（二分类）**
P(y|x) = pʸ · (1-p)^(1-y)
- y=1 → P=p；y=0 → P=1-p（一个式子统一两种情况）

## 二、核心推导

### 正态假设 → MSE
回归场景，假设误差 ε = y-ŷ 服从 N(0, σ²)：
```
P(y|x,w,b) = (1/√(2πσ²)) · exp( -(y-ŷ)²/(2σ²) )
取 log：
log P = log(1/√(2πσ²)) - (y-ŷ)²/(2σ²)
      = 常数 C - (y-ŷ)²/(2σ²)
n 个点：
log L = nC - Σ(yᵢ-ŷᵢ)²/(2σ²)
最大化 log L ⟺ 最小化 Σ(yᵢ-ŷᵢ)² = MSE
```

### 伯努利假设 → 交叉熵
二分类，p = 预测正类概率：
```
P(y|x) = pʸ · (1-p)^(1-y)
取 log：
log P = y·log(p) + (1-y)·log(1-p)
n 个点：
log L = Σ [ yᵢ·log(pᵢ) + (1-yᵢ)·log(1-pᵢ) ]
最大化 log L ⟺ 最小化 -Σ[...] = 交叉熵
```

## 三、核心 insight：统一框架

| 场景 | 概率假设 | MLE 推出的损失 |
| --- | --- | --- |
| 回归 | 误差服从正态分布 | MSE |
| 二分类 | 服从伯努利分布 | 交叉熵 |

同一个 MLE 框架，换分布假设 → 换损失函数。**损失函数是概率假设的必然结果，不是拍脑袋定的。** 以后遇到新任务，先想"数据/误差服从什么分布"，对应损失自然出来。

**变量 = 要优化的参数**：回归里是 w,b；σ 是数据噪声的固有属性，当常数（不训练它）。

## 四、我的误解与纠正（个性化）

| 我的误解 | 正确 |
| --- | --- |
| MLE 是"找极大值" | MLE 是找让"似然(观测数据出现的概率)"最大的"参数"——漏了"参数"和"似然"两个关键词 |
| 推导出 RMSE（均方根误差） | 推导出 MSE（均方误差）；RMSE = √(MSE) 是评估指标，不是推导出的损失 |
| "数据"服从正态分布 | 是"预测误差 ε=y-ŷ"服从正态，不是数据本身；y 可以任意分布 |
