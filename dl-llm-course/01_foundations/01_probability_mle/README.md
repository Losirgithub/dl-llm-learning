# L1.1 概率与统计视角

## 学习目标
- 理解随机变量、期望、方差、概率分布
- 掌握极大似然估计（MLE）的核心思想
- **核心成果**：从 MLE 推导出均方误差(MSE)和交叉熵损失，理解"为什么用这些损失函数"

## 前置依赖（前置门控）
- [ ] 微积分：用导数求函数极值
- [ ] 线性代数：向量/矩阵基本运算

## 理论要点
- 概率分布：离散(伯努利/多项式) vs 连续(高斯)
- 期望、方差
- 极大似然估计 MLE：给定数据，找最可能产生这些数据的参数
- MLE → 损失函数：高斯噪声假设→MSE；分类→交叉熵
- 贝叶斯视角（简介）

## 必需第三方库
- numpy：数值计算
- matplotlib：可视化分布
- scipy.stats：概率分布函数

## 掌握判定标准（检查点）
- 能说清 MLE 在做什么
- 能从高斯噪声假设推出 MSE
- 能从伯努利分布推出二元交叉熵
